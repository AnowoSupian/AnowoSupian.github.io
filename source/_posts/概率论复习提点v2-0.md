> 加法公式

**对于任意两个事件A，B，有：**

$$P(A∪B) = P(A) + P(B) - P(AB)$$

Plus : $P(A∪B∪C) = P(A) + P(B) + P(C) - P(AB) - P(BC) - P(AC) - P(ABC)$

> 减法公式

**设A，B为两个事件，则有：**

$$P(B-A) = P(B) - P(AB)$$

> 条件概率

设 $A，B$ 为两个事件，且 $P(B) > 0$ ， 则称  $\frac{P(AB)}{P(B)}$ 为事件 $B$ 已发生的条件下事件 $A$ 发生的条件概率，记为 $P(A|B)$ ，即

$$ P(A|B) = \frac{P(AB)}{P(B)} $$

$$ P(A_1∪A_2|B) = P(A_1|B) + P(A_2|B) - P(A_1B_1|B) $$

> 乘法公式

有条件概率的定义 $P(A|B) = \frac{P(AB)}{P(B)}$ ，可以得到 $P(AB) = P(A)P(B|A)$

> 全概率公式和贝叶斯公式

（全概率公式）设 $B$ 为样本空间 $Ω$ 中的任一事件， $A_1,A_2,...,A_n$ 为 $Ω$ 的一个划分，且 $P(A_i) > 0, i = 1,2,...,n,$ 则有

$$P(B) = P(A_1)P(B|A_1) + P(A_1)P(B|A_1) + ... + P(A_n)P(B|A_n) = \sum_{i=1}^{n} P(A_i) \times P(B|A_i)
$$

（贝叶斯公式）设 $B$ 为样本空间 $Ω$ 中的任一事件，$A_1,A_2,...,A_n$ 为 $Ω$ 的一个划分， 且 $P(B) > 0, P(A_i) > 0 , i = 1,2,...,n,$ 则有

$$P(A_i|B) = \frac{P(A_i)P(B|A_i)}{\sum_{j=1}^{n} P(A_j) \times P(B|A_j)}$$

$$
F(x) = \int_{-\infty}^{x} f(t) \, \mathrm{d}t
$$

则称 $X$ 为连续型随机变量，其中 $f(x)$ 称为 $X$ 的概率密度函数。连续型随机变量 $X$ 的分布函数 $F(x)$ 是连续函数。其具有以下性质

1° $f(x) \ge 0$

2° $\int_{-\infty}^{+\infty} f(t) \, \mathrm{d}t = 1$

3° $P$ {$x_1 < X \le x_2$} = $F(x_2) - F(x_1) = \int_{x_1}^{x_2} f(x) \, \mathrm{d}x$ $(x_1 \le x_2)$

4° 若 $f(x)$ 在点 $x$ 处连续，那么 $F`(x) = f(x)$

**一些重要推导性质：**

$P$ {$a \le X < b$} = $P$ {$a < X \le b$} = $P$ {$a \le X \le b$} = $P$ {$a < X < b$}

**其中 $P$ {$X = a$} 是几乎不可能发生的事件，它是“零概率事件”而非不可能事件**

4. 随机变量数字特征，求连续型随机变量函数的期望 （此题难点在于积分）

连续型随机变量X的概率密度为 $f(x)$

$$
E(x) = \int_{-\infty}^{+\infty} xf(x) \, \mathrm{d}x
$$

$$
E(Y=g(x)) = \int_{-\infty}^{+\infty} [Y=g(x)]f(x) \, \mathrm{d}x
$$

数学期望简称**期望**，又称为**均值**。

掌握 $E(XY)$ 那么 $E(X) = E(XY^0), E(Y) = E(X^0Y)$

**期望的性质：**

1° $E(c) = c$，其中 $c$ 为常数

2° $E(cX) = cE(x)$， 其中 $c$ 为常数

3° $E(X + Y) = E(X) + E(Y)$

4° 若 $X , Y$ 相互独立，则有 $E(XY) = E(X)E(Y)$

5. 切比雪夫不等式 - 大数定律

设随机变量 $X$ 的数学期望为 $E(X)$, 方差为 $D(X)$, 则对于任意正数 $\epsilon$ ，有：

$$ P \{ |X - E(X)| \ge \epsilon \} \le \frac{D(x)}{\epsilon^2}$$

> 两点分布

若随机变量 $X$ 只可能取 $x_1$ 与 $x_2$ 两个值，它的分布律为：

$$P\{X = X_1\} = 1 - p    (0 < p < 1)$$

$$P\{X = x_2\} = p$$

**特别地，当 $x_1 = 0, x_2 = 1$ 时，两点分布也叫做 $(0-1)$ 分布，记作 $X~(0-1)$ 分布**。

期望：$E(X) = p$

方差：$D(X) = p(1-p)$

> 二项分布

若随机变量 $X$ 的分布律为：$P\{X = k\} = C_n^k p^k(1-p)^{n-k}, 0 < p < 1, k = 0 , 1, 2, ..., n,$ 则称 $X$ 服从参数为 $n,p$ 的二项分布，记作 $X \sim b(n,p)$

期望：$E(X) = np$

方差：$D(X) = np(1-p)$

> 泊松分布

若随机变量 $X$ 的分布律为

$$P\{X = k\} = \frac{\lambda e^{-\lambda}}{k!}, k = 0,1,2,...,$$

其中 $\lambda > 0$ 是常数，则称 $X$ 服从参数为 $\lambda$ 的泊松分布，记为 $X \sim P(\lambda)$。

期望：$E(X) = \lambda$

方差：$D(X) = \lambda$

> 均匀分布

若随机变量 $X$ 具有概率密度

$$
F(x) = 
\left\{
\begin{array}{rcl} % r：第一列右对齐；c：第二列居中对齐（放等号）；l：第三列左对齐
    \frac{1}{b-a}, a < x < b \\
    \\
    \text{0}.
\end{array}
\right.
$$

则称 $X$ 在区间 $(a,b)$ 上服从均匀分布，记为 $X \sim U(a,b)$

期望：$E(X) = \frac{a+b}{2}$

方差：$D(X) = \frac{(b-a)^2}{12}$

> 指数分布

若随机变量 $X$ 的概率密度为

$$
f(x) = 
\left\{
\begin{array}{rcl} % r：第一列右对齐；c：第二列居中对齐（放等号）；l：第三列左对齐
    \lambda e^{-\lambda x}, a < x < b \\
    \\
    \text{0}.
\end{array}
\right.
$$

其中 $\lambda > 0$ 为常数，则称 $X$ 服从参数为 $\lambda$ 的指数分布， 记作 $X \sim E(\lambda)$。

期望：$E(X) = \frac{1}{\lambda}$

方差：$D(X) = \frac{1}{\lambda ^2}$

**正态分布的性质：**

1° 曲线关于直线 $x = \mu$ 对称

2° 曲线在 $x = \mu$ 处取到最大值

3° 曲线在 $x = \mu \pm \sigma$ 处有拐点

4° $\mu$ 为位置参数 $\sigma$ 为精度参数

**特别的，当$\mu = 0 ,\sigma = 1$ 时，称 $X$ 为标准正态分布**

期望：$E(X) = \mu$

方差：$D(X) = \sigma ^ 2$
